# RAID OF TITAN × THIRD ダイバーアプリ モック

RAID OF TITAN（UNCHAIN旗艦）に対応する **THIRD（配信介入プラットフォーム）側のダイバーアプリ** UI/UXモック。
Unityビルド前のイメージ共有・仕様すり合わせ用。

## 開き方

`index.html` をブラウザで開くだけ（依存ゼロ／Tailwind CDN）。
スマホ縦型（375×812）想定だが、PC ブラウザでも中央にスマホ枠で表示される。

```
open index.html
```

## 含まれる画面

| 画面 | 概要 |
|---|---|
| **所属国選択** | 初回ログイン時。フロスティア／インフェリア／ガイアスの3国から選ぶ（変更不可） |
| **ホーム** | 自国民アバター（クオータービュー）一覧。LIVE中／OFFLINE中／被攻撃中を一目で把握 |
| **介入（視聴）** | LIVE中はTPS視点（実機スクショ）／OFFLINE中は俯瞰ジオラマに自動切替。エリクサー＋手札4枚＋カード使用時の配置ゴースト表示 |
| **編成** | デッキ4枚（モック）＋所持カード一覧＋レベル＆ストック表示 |
| **遠征** | 進行中の遠征表示＋敵国アバターから攻撃目標選択（氷／土タブ） |
| **ショップ** | 買い切り（ブーストパック・デイリースペシャル）＋宝箱（実質ガチャ）。通貨はTHIRDポイント（THP）統合 |
| **設定** | 通知トグル／音量／アカウント |
| **宝箱モーダル** | 起動時に確率で立ち上がり。タップで開封＋カード獲得演出＋レベルアップ通知 |

## 仕様の核（モック共通）

### ダイバーの立場
- **自国（所属国）** ＝ 支援オンリー（協力型）
- **敵国** ＝ 妨害オンリー（遠征攻撃）

### カード（モック用4枚）
| カード | コスト | 役割 |
|---|---|---|
| 修理マン 🔧 | 4 | 施設修理 |
| ウルフマン 🐺 | 5 | 強攻撃役 |
| トルネードマン 🌪 | 6 | 巨人後退魔法 |
| 剣士マン ⚔ | 3 | 雑兵召喚 |

本実装時は固有名詞へ置換予定（Cannelloni／Lupo／Espresso などイタリア料理系統一）。

### 経済 / マネタイズ
- カード入手：初回ログイン無料パック ＋ **12時間ごと自動配布の宝箱** ＋ ガチャ／買い切り
- 同種カード重ねでレベルアップ（クラロワ式）
- 配信中の使用：**エリクサー（時間チャージ）** で消費。投げ銭概念なし
- ショップ通貨：**THIRDポイント（THP）** 統合（有償／無償の合算表示）
- ブーストパック：ログイン3日以内向け期間限定

### マルチプレイ構造
- **同時参加無制限**（視聴者全員がダイバー）
- **重ねがけ可**（100人が同じカードを切れば100体召喚＝視聴者数＝戦力）
- **3すくみ**：プレイヤー（PvE） × 自国ダイバー × 敵国ダイバー
- **遠征**：敵国プレイヤー1人を選んで非同期攻撃。自国も常時敵国ダイバーから狙われる

### 視点
- LIVE中：TPS視点（プレイヤーカメラ準拠）
- OFFLINE中：俯瞰カメラ（ジオラマ風、緩やかに揺れ）

### UIコンセプト
- スマホ縦型ベース（横転対応は本実装で）
- ホーム中央のアバターは **クオータービュー**（クラロワ／アメピグ等準拠）

## モックでの簡略事項
- 8枚デッキ→**4枚デッキ**（=デッキ全部が手札）
- 手札ローテーション概念なし
- 実機キャラ・建物は絵文字＋色／グラデーションで代替（本実装で3D化）
- 各種アニメは最低限
